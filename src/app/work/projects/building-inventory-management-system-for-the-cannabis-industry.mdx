## Overview

Trackr is an **inventory management application** built specifically for the cannabis industry, providing **real-time tracking**, **detailed item management**, and comprehensive reporting features. Developed with a modern tech stack including **Next.js 14**, **Clerk Auth**, **Tailwind CSS**, and **Prisma**, the app aims to combine **robust security** with **intuitive usability**, allowing cannabis businesses to **efficiently manage** their inventory processes. By offering **real-time inventory tracking** and **insightful analytics**, Trackr is designed to meet the **unique regulatory** and **operational needs** of the **cannabis sector**.


## Key Features

- **Order and Return Tracking**: Developed a system to track order and return statuses, updating inventory levels in real-time. This integration ensured accurate stock management and improved customer satisfaction.
- **Inventory Item Management**: Implemented an automated inventory item tracking system, allowing easy addition, editing, and deletion of items. This streamlined backend processes, enabling faster inventory updates with minimal manual intervention.
- **QR Code Scanner**: Integrated a QR code scanning feature for quick inventory checks and real-time updates, reducing manual entry errors and improving efficiency.
- **Scalable Interface**: Built a responsive, scalable interface using Tailwind CSS and ShadCN UI, ensuring consistency across devices and easy expansion as the application grows.

## Technologies Used

- **Next.js 14 and React**: These technologies were used for building the front-end codebase, enabling the creation of clean, reusable components that provide a seamless and responsive user experience. The combination of React’s component-based architecture with Next.js’s server-side rendering capabilities helped in achieving optimal performance and SEO.
- **Tailwind CSS**: Utilized for managing styles dynamically, ensuring a consistent and visually appealing UI throughout the application. It allowed the development team to build custom, responsive designs quickly while maintaining design consistency.
- **Clerk Authentication**: Clerk was integrated to handle secure user authentication, allowing users to log in and manage inventory securely. By using Clerk, Trackr ensures that user data remains protected, providing a seamless yet highly secure onboarding experience for users.
- **Prisma**: Used to manage database interactions, specifically to handle complex data relationships between growers, products, and transactions. Prisma’s schema-driven approach allowed for clear modeling of business requirements, while simplifying database operations and ensuring data integrity.
- **PostgreSQL on Vercel**: Trackr uses PostgreSQL as its primary database, hosted on Vercel, to ensure scalable and reliable backend data storage. This setup provides high availability and helps maintain data integrity while scaling with the application’s growing needs.
- **GitHub Actions**: Utilized for automating development workflows, including testing, building, and deploying updates to the application. This has streamlined the development process and helped maintain a high quality of code.

## Challenges and Learnings

The main challenges were managing **complex relational data**, securing **user authentication**, and maintaining **high performance** for **real-time data updates**. **Prisma**'s **schema capabilities** allowed effective modeling of relationships between **growers**, **products**, and **transactions**. **Clerk Auth** was used for **seamless, secure user flows**, while **Next.js** optimizations ensured **efficient data fetching** and **system responsiveness** under **heavy load**.

## Outcome

Trackr successfully provides the ability to **efficiently track inventory, manage orders, and handle returns**. The **intuitive UI**, combined with **QR code implementation** for **quick product scanning** and **tracking**, has led to greater operational transparency, efficiency, and ease of use. The **enhanced reporting features** provide actionable insights that allow businesses to make **informed decisions** about inventory levels, supplier performance, and overall **operational efficiency**. As a result, it is now **industry-proven** to be able to **reduce errors, streamline their inventory management processes, and save valuable time and resources**. These tangible benefits make Trackr a crucial tool for businesses in the cannabis industry.
